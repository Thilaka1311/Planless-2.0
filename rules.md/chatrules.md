# CHAT RULES

---

# Ⅰ. CHAT PHILOSOPHY

1. Chats exist only to support real-world plans.

2. Chats should feel:
- lightweight
- fast
- temporary
- coordination-first

3. Chats are NOT the primary product surface of Planless.

4. Chats should never evolve into:
- Discord-style communities
- permanent group chats
- endless messaging spaces

5. The purpose of chat is:
- coordination
- updates
- logistics
- real-time communication during plans

---

# Ⅱ. PLAN-BASED CHAT RULES

6. Every active plan should generate its own chat.

7. Chats must be tied directly to plans.

8. A new plan creates a new chat thread.

9. Chats should only remain active while the plan is active.

10. Once a plan ends:
- the chat should close
- the chat becomes read-only or archived

11. Old chats should not merge into future plans.

12. Every plan starts with a fresh coordination space.

---

# Ⅲ. CHAT EXPERIENCE RULES

13. Chat UI should remain:
- simple
- minimal
- easy to scan
- mobile-first

14. Latest messages should always appear at the bottom.

15. Time indicators should remain visible and lightweight.

16. Avoid:
- complex thread systems
- nested replies
- channels
- forums
- message reactions overload

17. Chat interactions should feel:
- quick
- responsive
- frictionless

18. Messaging should prioritize:
- speed
- readability
- coordination clarity

---

# Ⅳ. CHAT FEATURES RULES

19. Core MVP chat features may include:
- text messages
- image sharing
- lightweight reactions
- location sharing
- plan updates

20. Avoid adding during MVP:
- voice channels
- disappearing media systems
- AI chat features
- advanced moderation systems
- large media systems

21. Plan-related updates should remain visually prioritized.

Examples:
- venue changes
- timing updates
- cancellations
- confirmations

---

# Ⅴ. SOCIAL & BEHAVIOR RULES

22. Chats should support:
- participation
- coordination
- social momentum

23. Chats should not encourage:
- excessive messaging
- off-topic discussion
- long-term social dependency

24. The chat experience should feel similar to:
- WhatsApp group coordination
- event-based messaging

25. The emotional feeling of chat should be:
“We’re coordinating something happening in real life.”
